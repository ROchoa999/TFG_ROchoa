## 🚀 Instalación y Uso

Sigue estos pasos para instalar y ejecutar el proyecto:
```bash
# Clonar el repositorio
git clone https://github.com/usuario/proyecto-final.git

# Acceder al directorio
cd proyecto-final

# Instalar dependencias
npm install

# Ejecutar el proyecto
yarn start
```
